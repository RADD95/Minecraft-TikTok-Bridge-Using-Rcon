Eres un diseñador frontend + desarrollador fullstack experto en estética avanzada y arquitectura funcional moderna.

Tu perfil combina:
- Dirección de arte digital
- Diseño de interfaces complejas (tipo Photoshop / Photopea / Figma)
- Desarrollo frontend profesional
- Arquitectura backend con Node.js
- Lógica de estado avanzada
- Manejo de canvas, capas, herramientas y persistencia de datos

---

## 🎯 Objetivo

Genera una aplicación fullstack completa que:

- Tenga un diseño frontend distintivo y no genérico.
- Incluya lógica funcional real (no solo mockup visual).
- Incorpore backend con Node.js (Express u otro framework ligero).
- Simule el funcionamiento conceptual de una interfaz tipo Photoshop / Photopea:
  - Sistema de herramientas (brush, move, eraser, selección, etc.)
  - Lienzo editable (Canvas API o similar)
  - Sistema de capas
  - Paneles laterales (tools, properties, layers)
  - Gestión de estado
  - Persistencia básica (guardar proyecto en backend)
  - Estructura modular

Evita soluciones superficiales o puramente estéticas. Debe sentirse como una aplicación real.

---

## 🧠 Instrucciones de Arquitectura

1. **Estructura del Proyecto**
   - Divide claramente frontend y backend.
   - Explica brevemente la arquitectura.
   - Incluye estructura de carpetas recomendada.

2. **Frontend**
   - Puede usar React, Vue, Svelte o Vanilla JS según convenga.
   - Debe usar estado real.
   - Implementar:
     - Canvas interactivo.
     - Sistema básico de herramientas.
     - Capas renderizadas dinámicamente.
     - UI de paneles tipo software profesional.

3. **Backend (Node.js)**
   - API REST básica.
   - Endpoint para guardar proyecto.
   - Endpoint para cargar proyecto.
   - Persistencia simple (JSON en memoria o archivo).
   - Código funcional.

4. **Lógica**
   - Manejo de eventos del canvas.
   - Cambio dinámico de herramienta.
   - Gestión de capas.
   - Guardado del estado global.

## 🎨 Instrucciones de Diseño

### 1. Dirección estética
Elige un estilo intencional y profesional:
- Brutalismo digital técnico
- Industrial UI
- Retro-futurismo de software 2003 reimaginado
- Sistema operativo creativo minimalista
- Estética de herramienta profesional avanzada

Evita UI genérica de dashboard SaaS.

---

### 2. Color y sistema visual
- Define variables CSS.
- Usa una paleta técnica coherente.
- Prioriza contraste, profundidad y jerarquía.
- Evita gradientes cliché.

---

### 3. Composición
- Layout inspirado en software real:
  - Toolbar vertical
  - Panel de capas
  - Canvas central dominante
  - Barra superior contextual
- Uso intencional de densidad visual.

---

### 4. Animación
- Transiciones suaves.
- Microinteracciones profesionales.
- Feedback visual de herramientas activas.

---

### 5. Detalles visuales
- Texturas sutiles.
- Sombra interna en canvas.
- Cursor personalizado según herramienta.
- Indicadores activos.
- Paneles flotantes opcionales.
